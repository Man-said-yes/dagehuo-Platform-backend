# 搭伙平台后端接口文档

## 基础信息

- **服务地址**: `http://localhost:8080`
- **基础路径**: `/api`
- **Swagger文档**: `http://localhost:8080/swagger-ui.html`

---

## 接口概览

### 认证管理接口 (`/api/auth`)
- `GET /api/auth/test` - 服务连通性测试
- `POST /api/auth/login` - 微信登录
- `POST /api/auth/bind` - 绑定学号

### 2. 用户管理接口 (`/api/user`)
- `GET /api/user/info` - 获取当前用户信息
- `GET /api/user/info/{targetUserId}` - 获取指定用户信息 管理员接口
- `PUT /api/user/info` - 更新当前用户信息
- `PUT /api/user/nickname` - 修改当前用户昵称
- `PUT /api/user/credit/{targetUserId}` - 更新用户信誉分 管理员接口

### 活动管理接口 (`/api/event`)
- `POST /api/event` - 发布活动
- `GET /api/event/{eventId}` - 获取活动详情 
- `GET /api/event/list` - 获取活动列表
- `GET /api/event/my` - 获取我的活动
- `GET /api/event/participated` - 获取我参与的活动
- `PUT /api/event/{eventId}` - 更新活动
- `DELETE /api/event/{eventId}` - 删除活动
- `POST /api/event/{eventId}/join` - 加入活动
- `POST /api/event/{eventId}/exit` - 退出活动

## 配置说明

### 1. 微信小程序配置

在 `application.yaml` 中配置：

```yaml
wechat:
  app-id: your_app_id_here    # 替换为你的微信小程序appId
  app-secret: your_app_secret_here  # 替换为你的微信小程序appSecret
  test-mode: true  # 测试模式：true-使用code作为openid（开发测试用），false-调用真实微信接口

jwt:
  secret: dagehuo-platform-secret-key-2024  # JWT密钥
  expiration: 86400000  # Token过期时间（毫秒，默认24小时）
```

### 2. 测试模式说明

开发测试时，可以开启测试模式，无需真实的微信小程序配置：

- **test-mode: true** - 测试模式：code 直接作为 openid 使用
- **test-mode: false** - 生产模式：调用真实微信接口获取 openid

**测试模式使用方法**：
调用登录接口时，code 可以填任意字符串，例如：
- `test_openid_001`
- `user_123`
- `demo_user`

#### 1.2 微信登录

**接口地址**: `POST /api/auth/login`

**描述**: 通过微信code登录，返回JWT token和用户信息

**请求头**:
```
Content-Type: application/json
```

**请求参数**:

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| code | String | 是 | 微信登录code（测试模式下可填任意字符串） | test_user_001 |

**请求示例**:
```json
{
  "code": "test_user_001"
}
```

**测试模式请求示例**（推荐）:
```json
{
  "code": "demo_user_123"
}
```

**响应示例**:

**老用户（已绑定学号）**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "registered": true,
    "studentId": "24000000000",
    "userId": 1
  }
}
```

**新用户（未绑定学号）**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "registered": false,
    "studentId": null,
    "userId": 2
  }
}
```

**失败响应**:
```json
{
  "code": 0,
  "msg": "登录失败: 获取微信openid失败",
  "data": null
}
```

**响应数据说明**:

| 字段 | 类型 | 说明 |
|------|------|------|
| token | String | JWT令牌，后续接口需要在请求头中携带 |
| registered | Boolean | 用户是否已绑定学号 |
| studentId | String | 学号（如果已绑定） |
| userId | Long | 用户ID |

## 前端调用示例

### 1. 微信小程序登录流程

```javascript
// 1. 调用微信登录接口获取code
wx.login({
  success: (res) => {
    if (res.code) {
      // 2. 将code发送到后端
      wx.request({
        url: 'http://localhost:8080/api/auth/login',
        method: 'POST',
        data: {
          code: res.code
        },
        success: (response) => {
          if (response.data.code === 1) {
            const { token, registered, studentId, userId } = response.data.data;
            
            // 3. 保存token
            wx.setStorageSync('token', token);
            
            // 4. 判断是否需要绑定学号
            if (registered) {
              // 已绑定，跳转到首页
              wx.switchTab({ url: '/pages/index/index' });
            } else {
              // 未绑定，跳转到绑定页面
              wx.navigateTo({ url: '/pages/bind/bind' });
            }
          }
        }
      });
    }
  }
});
```

### 2. 绑定学号

```javascript
// 从token中获取openid，或者直接使用登录时保存的
const openid = '...';

wx.request({
  url: 'http://localhost:8080/api/auth/bind',
  method: 'POST',
  data: {
    openid: openid,
    studentId: '24000000000'
  },
  success: (response) => {
    if (response.data.code === 1) {
      wx.showToast({
        title: '绑定成功',
        icon: 'success'
      });
      // 跳转到完善信息页面
      wx.navigateTo({ url: '/pages/profile/profile' });
    } else {
      wx.showToast({
        title: response.data.msg,
        icon: 'none'
      });
    }
  }
});
```

### 3. 完善用户信息

```javascript
// 登录成功后保存的userId
const userId = 1;

// 调用获取用户信息接口
wx.request({
  url: 'http://localhost:8080/api/user/info/' + userId,
  method: 'GET',
  success: (response) => {
    if (response.data.code === 1) {
      const userInfo = response.data.data;
      // 显示用户信息到页面
      console.log('用户信息:', userInfo);
    }
  }
});

// 更新用户信息
wx.request({
  url: 'http://localhost:8080/api/user/info/' + userId,
  method: 'PUT',
  data: {
    nickname: '张三',
    avatar: 'https://example.com/avatar.jpg',
    gender: 1,
    phone: '13800138000'
  },
  success: (response) => {
    if (response.data.code === 1) {
      wx.showToast({
        title: '保存成功',
        icon: 'success'
      });
      // 跳转到首页
      wx.switchTab({ url: '/pages/index/index' });
    } else {
      wx.showToast({
        title: response.data.msg,
        icon: 'none'
      });
    }
  }
});
```

## 注意事项

1. **跨域支持**: 所有接口已配置 `@CrossOrigin(origins = "*")`，支持跨域请求
2. **学号格式**: 必须以 "24" 开头，且为11位数字
3. **学号唯一性**: 同一学号只能被一个微信账号绑定
4. **参数验证**: 所有必填参数都会进行格式验证
5. **Token有效期**: 默认24小时，过期需要重新登录
6. **微信配置**: 需要在 application.yaml 中正确配置 appId 和 appSecret
7. **活动权限**: 只有活动创建者可以更新或删除活动
