# 搭伙平台后端接口文档

## 基础信息

- **服务地址**: `http://localhost:8080`
- **基础路径**: `/api`
- **Swagger文档**: `http://localhost:8080/swagger-ui.html`

---

## 配置说明

### 1. 微信小程序配置

在 `application.yaml` 中配置：

```yaml
wechat:
  app-id: your_app_id_here    # 替换为你的微信小程序appId
  app-secret: your_app_secret_here  # 替换为你的微信小程序appSecret

jwt:
  secret: dagehuo-platform-secret-key-2024  # JWT密钥
  expiration: 86400000  # Token过期时间（毫秒，默认24小时）
```

## 统一返回格式

所有接口的返回结果统一使用以下格式：

```json
{
  "code": 1,
  "msg": "success",
  "data": {}
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| code | Integer | 状态码：1成功，0失败 |
| msg | String | 响应信息 |
| data | Object | 返回数据 |

## 接口列表

### 1. 认证管理接口 (`/api/auth`)

#### 1.1 服务连通性测试

**接口地址**: `GET /api/auth/test`

**描述**: 测试服务是否正常运行

**请求参数**: 无

**响应示例**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "message": "服务正常",
    "timestamp": 1234567890123
  }
}
```

---

#### 1.2 微信登录

**接口地址**: `POST /api/auth/login`

**描述**: 通过微信code登录，返回JWT token和用户信息

**请求头**:
```
Content-Type: application/json
```

**请求参数**:

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| code | String | 是 | 微信登录code | 023abc123def456 |

**请求示例**:
```json
{
  "code": "023abc123def4444"
}
```

**响应示例**:

**老用户（已绑定学号）**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "registered": true,
    "studentId": "24000000000",
    "userId": 1
  }
}
```

**新用户（未绑定学号）**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "registered": false,
    "studentId": null,
    "userId": 2
  }
}
```

**失败响应**:
```json
{
  "code": 0,
  "msg": "登录失败: 获取微信openid失败",
  "data": null
}
```

**响应数据说明**:

| 字段 | 类型 | 说明 |
|------|------|------|
| token | String | JWT令牌，后续接口需要在请求头中携带 |
| registered | Boolean | 用户是否已绑定学号 |
| studentId | String | 学号（如果已绑定） |
| userId | Long | 用户ID |

---

#### 1.3 绑定学号

**接口地址**: `POST /api/auth/bind`

**描述**: 将微信openid与学号绑定

**请求头**:
```
Content-Type: application/json
```

**请求参数**:

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| openid | String | 是 | 微信openid | o1234567890abcdef |
| studentId | String | 是 | 学号，24开头11位数字 | 24000000000 |

**请求示例**:
```json
{
  "openid": "o1234567890abcdef",
  "studentId": "24000000000"
}
```

**响应示例**:

**成功**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "studentId": "24000000000"
  }
}
```

**失败响应**:
```json
{
  "code": 0,
  "msg": "学号格式不正确（24开头11位数字）",
  "data": null
}
```

```json
{
  "code": 0,
  "msg": "该学号已被绑定",
  "data": null
}
```

```json
{
  "code": 0,
  "msg": "用户不存在",
  "data": null
}
```

## 前端调用示例

### 1. 微信小程序登录流程

```javascript
// 1. 调用微信登录接口获取code
wx.login({
  success: (res) => {
    if (res.code) {
      // 2. 将code发送到后端
      wx.request({
        url: 'http://localhost:8080/api/auth/login',
        method: 'POST',
        data: {
          code: res.code
        },
        success: (response) => {
          if (response.data.code === 1) {
            const { token, registered, studentId, userId } = response.data.data;
            
            // 3. 保存token
            wx.setStorageSync('token', token);
            
            // 4. 判断是否需要绑定学号
            if (registered) {
              // 已绑定，跳转到首页
              wx.switchTab({ url: '/pages/index/index' });
            } else {
              // 未绑定，跳转到绑定页面
              wx.navigateTo({ url: '/pages/bind/bind' });
            }
          }
        }
      });
    }
  }
});
```

### 2. 绑定学号

```javascript
// 从token中获取openid，或者直接使用登录时保存的
const openid = '...';

wx.request({
  url: 'http://localhost:8080/api/auth/bind',
  method: 'POST',
  data: {
    openid: openid,
    studentId: '24000000000'
  },
  success: (response) => {
    if (response.data.code === 1) {
      wx.showToast({
        title: '绑定成功',
        icon: 'success'
      });
      // 跳转到首页
      wx.switchTab({ url: '/pages/index/index' });
    } else {
      wx.showToast({
        title: response.data.msg,
        icon: 'none'
      });
    }
  }
});
```

## 注意事项

1. **跨域支持**: 所有接口已配置 `@CrossOrigin(origins = "*")`，支持跨域请求
2. **学号格式**: 必须以 "24" 开头，且为11位数字
3. **学号唯一性**: 同一学号只能被一个微信账号绑定
4. **参数验证**: 所有必填参数都会进行格式验证
5. **Token有效期**: 默认24小时，过期需要重新登录
6. **微信配置**: 需要在 application.yaml 中正确配置 appId 和 appSecret
