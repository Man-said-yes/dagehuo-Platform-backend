# 搭伙平台后端接口文档

## 基础信息

- **服务地址**: `http://localhost:8080`
- **基础路径**: `/api`
- **Swagger文档**: `http://localhost:8080/swagger-ui.html`

---

## 统一返回格式

所有接口的返回结果统一使用以下格式：

```json
{
  "code": 1,
  "msg": "success",
  "data": {}
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| code | Integer | 状态码：1成功，0失败 |
| msg | String | 响应信息 |
| data | Object | 返回数据 |

---

## 接口列表

### 1. 认证管理接口 (`/api/auth`)

#### 1.1 服务连通性测试

**接口地址**: `GET /api/auth/test`

**描述**: 测试服务是否正常运行

**请求参数**: 无

**响应示例**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "message": "服务正常",
    "timestamp": 1234567890123
  }
}
```

---

#### 1.2 微信登录

**接口地址**: `POST /api/auth/login`

**描述**: 通过微信openid登录，返回用户注册状态

**请求头**:
```
Content-Type: application/json
```

**请求参数**:

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| openid | String | 是 | 微信openid | o1234567890abcdef |

**请求示例**:
```json
{
  "openid": "o1234567890abcdef"
}
```

**响应示例**:

**老用户（已绑定学号）**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "registered": true,
    "studentId": "24000000000"
  }
}
```

**新用户（未绑定学号）**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "registered": false
  }
}
```

**失败响应**:
```json
{
  "code": 0,
  "msg": "登录失败: xxx",
  "data": null
}
```

---

#### 1.3 绑定学号

**接口地址**: `POST /api/auth/bind`

**描述**: 将微信openid与学号绑定

**请求头**:
```
Content-Type: application/json
```

**请求参数**:

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| openid | String | 是 | 微信openid | o1234567890abcdef |
| studentId | String | 是 | 学号，24开头11位数字 | 24000000000 |

**请求示例**:
```json
{
  "openid": "o1234567890abcdef",
  "studentId": "24000000000"
}
```

**响应示例**:

**成功**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "studentId": "24000000000"
  }
}
```

**失败响应**:
```json
{
  "code": 0,
  "msg": "学号格式不正确（24开头11位数字）",
  "data": null
}
```

```json
{
  "code": 0,
  "msg": "该学号已被绑定",
  "data": null
}
```

---

## 数据模型

### LoginRequest（登录请求）

| 字段 | 类型 | 说明 |
|------|------|------|
| openid | String | 微信openid |

### BindRequest（绑定请求）

| 字段 | 类型 | 说明 |
|------|------|------|
| openid | String | 微信openid |
| studentId | String | 学号（24开头11位数字） |

### Result（统一返回结果）

| 字段 | 类型 | 说明 |
|------|------|------|
| code | Integer | 状态码：1成功，0失败 |
| msg | String | 响应信息 |
| data | Object | 返回数据 |

---

## 错误码说明

| code | 说明 |
|------|------|
| 1 | 成功 |
| 0 | 失败 |

---

## 注意事项

1. **跨域支持**: 所有接口已配置 `@CrossOrigin(origins = "*")`，支持跨域请求
2. **学号格式**: 必须以 "24" 开头，且为11位数字
3. **学号唯一性**: 同一学号只能被一个微信账号绑定
4. **参数验证**: 所有必填参数都会进行格式验证
